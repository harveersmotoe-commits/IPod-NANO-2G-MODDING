# IPod-NANO-2G-MODDING
Modding an old ipod nano 2nd generation (New firmware, Battery and USB-C)

Battery (IFixIt): https://www.ifixit.com/Guide/iPod+Nano+2nd+Generation+Battery+Replacement/422
